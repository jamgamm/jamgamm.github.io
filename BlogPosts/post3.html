Hit the Slay Button

May 30, 2022


A project I did involving push buttons and the Arduino Leonardo. This is just a little preview of one of three buttons I made. This project was inspired by a tiktok that would press alt+f4 whenever the button was pressed. I thought it would be fun to have a button that types "slay" when it was pressed so I decided to bring it to life!

You can check out the full details and my code for this on my github here.
